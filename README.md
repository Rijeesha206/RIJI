# RIJI - Data Science & ML Projects

Collection of machine learning and data analysis projects built during my internship and coursework.

## Projects

### 1. Parkinson's Disease Detection
Predicts Parkinson's disease using speech data and ML models.  
**Tech**: Python, Scikit-learn, Pandas  
**File**: `Project_14_Parkinson's_Disease_Detection.ipynb`

### 2. Car Price Prediction
Regression model to predict car prices based on features.  
**Tech**: Python, Scikit-learn, Matplotlib  
**File**: `Project_8_Car_Price_Prediction.ipynb`

### 3. Breast Cancer Classification
Binary classification to detect malignant vs benign tumors.  
**Tech**: Python, Scikit-learn, Logistic Regression  
**File**: `breast_cancer.ipynb`

### 4. Titanic Survival Analysis
Exploratory data analysis and prediction of survival rates.  
**Tech**: Python, Pandas, Seaborn  
**File**: `Titanic_data.ipynb`

### 5. Spam/Ham Email Classifier
NLP model to classify emails as spam or ham.  
**Tech**: Python, NLTK, Scikit-learn  
**File**: `ham_mail_or_spam_mal.ipynb`

### 6. Gold Price Prediction
Time series analysis for gold price forecasting.  
**Tech**: Python, Pandas, Matplotlib  
**File**: `gold_price2.ipynb`

### 7. Concrete Strength Prediction
Regression model to predict concrete compressive strength.  
**Tech**: Python, Scikit-learn  
**File**: `concrete_prediction.ipynb`

### 8. AI Projects
Additional ML experiments and practice notebooks.  
**Files**: `AI.ipynb`, `AI2.ipynb`

## How to Run
1. Open any `.ipynb` file in Jupyter Notebook or Google Colab
2. Run cells sequentially
3. Install requirements if prompted

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, TensorFlow
